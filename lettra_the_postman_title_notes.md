# LETTRA the Postman — タイトル決定までのメモ

## ゲーム概要
ビブリボン的な横スクロールリズムゲーム。主人公は配達員で、リズムに合わせて道中のポストに手紙を投函していく。

## 最終決定
- **正式タイトル**: `LETTRA the Postman`
- **通称・ロゴ**: `LETTRA`(単独でも使う、二段運用)
- **構文の元ネタ**: Chance the Rapper / Tyler, the Creator 系の「固有名 + the + 役職」

## 決め手
1. **固有性** — `LETTRA the Postman` でググると Ryo のものしか出ない(ドメイン取得時点でゼロヒット確認済み)
2. **説明力** — `the Postman` の3単語でジャンル・主人公像・トーンが伝わる
3. **ロゴの強さ** — `LETTRA` 単体でロゴ化できる(下記)
4. **拡張性** — `the Captain`(船便)、`the Pilot`(航空便)、`the Pigeon`(伝書鳩)など続編構文を最初から内包

## 「LETTRA」という綴りの強み
- 6文字、全大文字、ディセンダー無し → ベースライン完全に揃う
- **中央の TT が二本柱** → ポスト2本・電柱2本・赤い旗2本に置換可能、ロゴ自体がアイコン化できる
- L と A の対角バックエンドで左右が安定
- イタリア語/ロマンス語的な響きで詩的、女性名・人名としても機能

## 「LETTRA」周辺ブランド地形(把握しておく事項)
ゲームでの直接衝突は無いが、`Lettra` 自体は手紙・印刷・タイポグラフィ界隈で既に名前が立っている:
- **Crane's Lettra** — 200年以上続くアメリカの製紙会社 Crane の高級レタープレス用紙ブランド(letterpress 界隈で有名)
- **PP Lettra Mono** — Pangram Pangram Foundry の書体、Olivetti Lettera 22 タイプライター由来
- **Lettra Press** — フィリピンの小規模出版社
- **Planet Lettra** — 古い iOS 子ども向けフォニックスアプリ

これらは同じ星座にいる(手紙・活版・タイポグラフィ)ので、デザイナー・ステーショナリー好きの海外層には**むしろプラスに作用する可能性**がある。`the Postman` の複合化により混同は回避される。

## 隠し札
**Lettrism(レトリスム)** — 1940年代パリの前衛芸術運動。Isidore Isou が興した、文字そのものを芸術の根源に据える運動で、詩・映画・絵画・音楽全部に展開された。LETTRA というタイトルとの語源的接続は、海外プレスや批評家向けインタビューで作家性を語る時の隠しフックになる。

## ロゴ方向(デザイナーへのブリーフ用メモ)

### コアアイデア
TT を主役にする。LETTRA の中央にある二本の T を「ポスト2本」「リズムの打点2発」のメタファーとして扱い、ブランドの記号にする。

### フルロゴ / コンパクトの二段運用
- **フルロゴ**: `LETTRA` を主役に、TT に何らかの介入(色/形/装飾)を加える。下か横に小さく `the Postman`
- **コンパクト版**: `TT` だけ切り出してアイコン化(Netflix の N、Tyler, the Creator の T のように)。SNS アイコン、ファビコン、Steam Library Icon 用

### 検討したコンセプト
1. 純粋ワードマーク(無加工、書体勝負)
2. セリフイタリック(詩的方向、in the blue shirt 寄り)
3. **TT アクセントカラー**(二本の T を別色 — 例: ポスト赤)← 推し
4. ポストマーク(円形消印)シール
5. 取消線(キャンセレーション)を文字に重ねる
6. **TT モノグラム**(コンパクト版 / アイコン用)← 推し

### 世界観の隣接物
- in the blue shirt の音楽性(静謐、リズム、青系の色調)
- 配達員=リズマー比喩(投函音=スネア、足音=キック)
- Lettrism の文字芸術的なミニマルさ
- Crane's Lettra の活版印刷の手触り
